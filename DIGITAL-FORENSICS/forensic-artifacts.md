# Forensic Artifacts

## What are Forensic Artifacts?

Forensic artifacts are traces of user or system activity left behind on a computer.

## Common Artifacts

- Browser history
- Recent files
- Event logs
- Registry entries
- Download history

## Why are They Important?

Artifacts help investigators reconstruct events and determine what actions occurred on a system.

## Key Point

Even if a file is deleted, artifacts may still provide evidence that it once existed.

## What I Learned

Forensic artifacts can reveal valuable information about user behaviour and system activity.
