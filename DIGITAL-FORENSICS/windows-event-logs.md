# Windows Event Logs

## What are Windows Event Logs?

Windows Event Logs record important operating system, security, and application events.

## Main Log Types

### Security

Records authentication events and security-related activity.

### System

Records operating system and hardware events.

### Application

Records events generated by installed applications.

## Why are They Important?

Event logs can help investigators identify suspicious behaviour and reconstruct events during an investigation.

## Key Point

Windows Event Logs are often one of the first places examined during a forensic investigation.

## What I Learned

Windows Event Logs provide valuable evidence about user activity, system changes, and security events.
