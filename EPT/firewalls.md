# Firewalls

This file contains notes on firewalls and their role in protecting systems and networks.

---

## What is a Firewall?

A firewall is a security device or software application that monitors and controls network traffic based on predefined security rules.

Its purpose is to prevent unauthorized access while allowing legitimate traffic.

---

## Types of Firewalls

### Network Firewall
Protects an entire network.

### Host-Based Firewall
Runs on an individual device.

Examples:
- Windows Defender Firewall
- Linux iptables

---

## Inbound vs Outbound Traffic

### Inbound Traffic
Traffic entering a network or device.

Example:
- A user visiting a web server.

### Outbound Traffic
Traffic leaving a network or device.

Example:
- A workstation communicating with a website.

---

## Why Firewalls Matter

Firewalls help:
- Reduce the attack surface
- Block unauthorized access
- Monitor network activity
- Prevent malicious connections

---

## Common Firewall Rules

Examples:
- Allow HTTPS (Port 443)
- Allow SSH (Port 22)
- Block unnecessary services
- Restrict remote access

---

## What I Learned

- Firewalls control traffic entering and leaving systems.
- They help reduce exposure to threats.
- Monitoring outbound traffic can help identify compromised devices.
